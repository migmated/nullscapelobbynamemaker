# Nullscape Lobby Name Maker

Making custom lobby names on Nullscape it's a hidden feature that is possible through Roblox's Rich Text, but doing it in hand is so boring and can take some of your time, thinking of that... I present for you Nullscape Lobby Name Maker! That makes the entire experience more visual and easy.

**THIS IS A FAN-MADE PROJECT, AND IT IS NOT OFFICIALLY AFILLIATED TO NULLSCAPE.**
